## mysandbox
this workspace was generated with the command 
```
 npx create-nx-workspace mysandbox --preset=next --appName=my-app
```
which created the app my-app at `mysandbox/apps/my-app`
to start the app
```
cd mysandbox
yarn start my-app
```

### adding first component to my-app

```
cd my-app
```
